# Ultimate Age Calculator 🗓️📅

A smart and precise Python program that calculates the user's exact age in years, months, and days. The project has been upgraded to support both **Hijri** and **Gregorian** calendars with an automated cross-conversion feature.

## ✨ Features
- **Hijri Calendar Support:** Calculates age based on the Hijri calendar (Umm al-Qura system).
- **Gregorian Calendar Support:** Calculates age based on the standard Gregorian calendar.
- **Smart Cross-Conversion:** - Inputting a Hijri date automatically displays your birth date in Gregorian.
  - Inputting a Gregorian date automatically displays your birth date in Hijri.
- **Interactive Menu:** Users can seamlessly choose their preferred calendar and perform multiple calculations without restarting the app.

## 🛠️ Prerequisites & Installation
This project requires Python 3 and the `hijri-converter` library.
